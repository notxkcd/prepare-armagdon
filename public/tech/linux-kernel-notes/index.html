Linux Kernel Memory Management

October 25, 2023 | linux, kernel, c

Understanding kmalloc vs vmalloc.

kmalloc allocates physically contiguous memory. It is faster but limited in size. vmalloc allocates virtually contiguous memory.

void *ptr = kmalloc(size, GFP_KERNEL);
if (!ptr)
    return -ENOMEM;
